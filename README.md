# Traxly 🔗

A full-stack link management & analytics platform built with the MERN stack.

## Features
- URL shortening with custom aliases
- QR code generation
- Link expiration & password protection
- Real-time analytics dashboard (clicks, devices, browsers, referrers)
- Redis caching for fast redirects
- Link-in-bio page builder

## Tech Stack
React.js · Node.js · Express.js · MongoDB · Redis · JWT · Tailwind CSS · Recharts
